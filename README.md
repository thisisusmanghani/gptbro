## Deploy

```bash
# Deploy to Vercel
vercel --prod

# Or push to GitHub (auto-deploy)
git push origin main
```

## Local Dev

```bash
# Install dependencies
cd frontend && npm install

# Run with Vercel dev server
cd .. && vercel dev
```
